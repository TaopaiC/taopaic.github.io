Google Toolbar 3 Beta (for Firefox)

| Comments


Google Toolbar Beta ( 官方 blog )

終於有新版了, 不但加入原本 IE 版有的功能

  • 自訂按鈕
  • 書籤同步
  • 帳號登入

而且如果是英文語系的 firefox , 還有..

  • 使用 Google Docs & Spreadsheets 開啟檔案
    可以開啟

    1. 網站上的檔案
    2. 拉 icon 進 firefox
    3. 雙擊該檔 (windows有, linux無此功能)

真是太棒了, firefox版和ie版的比起來, 一點都不陽春了…

如果是使用 ubuntu 的話, 可以參考 在 ubuntu edgy 上安裝 Google Toolbar 安裝, 至於換語系的話,

echo ‘export LANG=“en_US.UTF8″‘ >> ~/.mozilla/firefox/rc

就可以換成英文語系.
