Yahoo開發者社群中文版 (Yahoo! Developer Network)

| Comments

查資料時, 才驚覺上禮拜Yahoo!奇摩推出YDN中文版.

我比較感興趣的:

一個架在無名的官方YDN部落格 . (疑? 不是在Yahoo! Blog啊?)

YUI 的部份, 雖然還寫 2.3.1 版本 (現在有 2.4.1 版本), 但有全中文的 “首頁“ 和 “常見問題” , 其他文件還是連去美國YDN的YUI網頁.

另一個是台灣本土的 Yahoo!生活+ 開放 API 了, 有個超級簡陋的說明文件. 對了, 他的限制

Yahoo!生活+網路服務目前限制每天2000次，在於非商業性的使用。更多相關的使用限制，請參考使用規範的部份。

誰可以教我怎麼用每天2000次做個堪用的應用啊啊啊啊…

沒有地圖 API 說明文件, 這也不能怪台灣Y! , 美國 Y! 的地圖, 台灣部份根本沒有圖/影像, 看Y!生活+地圖 (網址是 tw.map.yahoo…) 都在使用 urmap 了. (Leeym的blog有較詳細的介紹 ). 也只能拿 Google Map 或是 UrMap 來用了.

怎麼整篇文章都在抱怨, Y!Kimo 推中文 YDN 總是好事.
