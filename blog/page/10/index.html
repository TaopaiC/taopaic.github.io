解決 Flash 文字無法顯示的問題 (on Ubuntu Edgy)

| Comments

最近發現用firefox看有flash的網頁總是怪怪的, 剛剛連去中國信託的網頁, 才發現flash裡面的文字無法顯示, 所以網頁看起來怪怪的… 選單看不到文字, 根本無法使用網路銀行… /_\
找了許多網頁, 都不外乎說要安裝 msttcorefonts 和 gsfonts-x11 , 前者是我必裝的字型套件, 後者是 flashplayer-nonfree 的依賴套件, 所以這兩個套件我都有安裝, 這個解法對我無效. 奇怪的是, 我的notebook使用同一版本的ubuntu linux, 差不多一樣的環境, 卻沒這個問題.

最後在 FlickrBugs 那邊找到解法, 提到另一種方法,

% cd /var/lib/defoma/x-ttcidfont-conf.d/dirs/TrueType/
% sudo ttmkfdir ./
% sudo mkfontdir ./

( 如果沒有 ttmkfdir 這個指令的話, 請安裝 ttmkfdir 套件 )

我就檢查桌機和notebook的該目錄, 確實就是桌機缺少 fonts.dir 這個檔案. 執行 mkfontdir 之後, flash 就可以顯示文字了~

參考資料:

  • The flash plugin and X.Org 7.0 (X11R7) font problems

代糖的甜點?

| Comments

上週末, 經過台北地下街 (台北新世界), 想順路去找之前看到好評的泡芙, 沒想到沒有賣泡芙的店. 看到一家賣像是布丁的東西, 就去看看. 他們賣的是代糖的甜點, 標榜糖尿病患者也能吃的甜點, 店員一直強調熱量不高. 不便宜, 所以只買兩個嘗嘗看.

剛剛吃了半個, 不錯吃, 但可惜不是我喜歡的那種口味. 不過我相信對不能吃糖的人, 這個一定很好吃.

ps. 回家後, 細讀他的傳單, 才發現他們的另一種目標對象是減重者. 我是因為好奇才買的, 不是因為減肥喔喔喔喔…
