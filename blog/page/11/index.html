| Comments

ubuntu 裝在 Acer TM3002 上面有不小的問題, 不能說是 ubuntu 的問題, 應該是 acer 的 bios 和 linux kernel 之間的問題

Ubuntu 是好東西

| Comments

ubuntu 真的是好東西, 不用去煩惱設定, 光碟放進去, 裝好就可以用了.

以前都是在 server 安裝 FreeBSD 或是 Gentoo (Linux系) , 兩者都有 ports , 可以調整 ports 的編譯選項等優點.
可是要拿他們當作桌機, XWindow 和 中文 的繁雜設定讓我卻步…

Linux 我只使用過 Gentoo 和 ubuntu 兩種, ubuntu 和其他家的 linux 優缺我並不清楚, 但是 ubuntu 有些特性我很喜歡

  • LiveCD安裝完, 就有圖形介面可以使用. 有圖形介面程式可以設定要使用的語言.
  • 每天都有10個以上的更新…. XD
  • 有圖形介面可以安裝軟體, 不用自己處理相依性

現在家裏的電腦已經使用 ubuntu 6 一個多月了, 雖然還不是穩定版 (穩定的版本是 ubuntu 5), 但使用上沒啥問題. 下次要安裝 MS windows 以外的作業系統時, 推薦你考慮一下ubuntu .

一銀網銀的(小?)錯誤

| Comments

剛剛線上申請一銀的網銀功能，只需要晶片卡讀卡機和晶片金融卡就可以申請，我申請完成後，按下網頁上導到網銀的網址，結果出現”無法顯示網頁” 的錯誤..
仔細一看，他的網址打錯了， firstbank 打成 furstbank，結果就導到 http://nb1.furstbank.com.tw/ 去了…

沒有申請成功網頁的抓圖，只有寄到信箱中的信件，不過錯誤還是一樣!

ps: 查 whois ，furstbank.com.tw 還沒有人申請… 應該不會有人做壞事吧 XD
